Hexo on Centos 6

发表于 2018-02-09

安装 Node.js 环境

1. 安装Node.js依赖包

yum -y install gcc-c++ openssl-devel

2. 检查Python版本>2.6

python –version

3. 下载和安装Node.js

【注】编译需要 gcc 4.8, 在centos6中安装方法如下：

参考：https://gist.github.com/giwa/b1fb1e44dc0a7d270881

4. 检查Node.js版本

node -v

部署和安装 Hexo 博客

1. 安装 Hexo

npm install -g hexo

2. 部署文件夹

mkdir hexo
cd hexo

3. 初始化 Hexo

hexo init

4. 安装依赖包

npm install

5. 生成静态页面

hexo g

6. 配置文件 _config.yml

deploy:
type:github
repository:git@coding.net:hexocoding/hexocoding.git
branch:master
